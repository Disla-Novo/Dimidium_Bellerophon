# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

Please report security vulnerabilities through the **"Report a vulnerability"** button in the Security tab of this repository.

You can expect an acknowledgment within 48 hours. If the vulnerability is accepted, we'll work on a fix and release it as soon as possible. If declined, we'll explain why.
